# Cargo Management System

## English

A database management system developed for cargo and shipment operations using SQL Server.

### Features
- Customer management
- Cargo tracking
- Shipment records
- Database relationships
- SQL queries and reporting

### Technologies
- SQL Server
- Relational Database Design

---

## Türkçe

SQL Server kullanılarak geliştirilmiş kargo ve gönderi işlemlerine yönelik bir veritabanı yönetim sistemi.

### Özellikler
- Müşteri yönetimi
- Kargo takibi
- Gönderi kayıtları
- Veritabanı ilişkileri
- SQL sorguları ve raporlama

### Teknolojiler
- SQL Server
- İlişkisel Veritabanı Tasarımı
